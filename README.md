# Sampark AI Platform

Sampark AI Decision Intelligence Platform — multi-agent, RAG-grounded system on Google Cloud.

## Overview
This platform provides an intelligent backend for community issue reporting and management, leveraging LangGraph, Vertex AI, and Google Cloud services.

## Setup
To install dependencies for development:
```bash
pip install -e ".[dev]"
```

## Testing
To run the test suite:
```bash
pytest
```
